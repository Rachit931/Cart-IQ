# Cart-IQ---E-commerce-Intelligence-System
An end-to-end machine learning system that analyzes customer behavior to predict purchases, categorize users, estimate spending, and generate personalized product recommendations.

## Problem Statement

E-commerce platforms struggle to understand customer behavior at scale:
- Who is likely to purchase ?
- Which customers are at risk of churn ?
- How much will a customer spend ?
- What products should be recommended ?

CartIQ solves these multiple problems by combining multiple machine learning approaches into a combined intelligence system.
